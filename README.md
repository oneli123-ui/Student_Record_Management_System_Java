# Student Record Management System (Java)

This project is a simple console-based Student Record Management System developed using Java.

## Features
- Add student records
- Edit student details
- Remove student records
- View all students
- Determine pass/fail status based on marks

## Technologies Used
- Java (OpenJDK 21)
- IntelliJ IDEA Community Edition
- GitHub for version control

## Purpose
This project was developed as part of CSP3341 to demonstrate core Java programming concepts, environment setup, and basic software design principles.
